## Diferenças vs v1 (que foi rejeitada)

| Aspecto | v1 (12 slides) | v2 (25 slides) |
|---|---|---|
| Quantidade | 12 slides — perdeu informação | 25 slides — preservou 100% do PDF |
| Timeline | 5 marcos (cortou 6) | 11 marcos divididos em 2 slides |
| Estrutura de gestão | só 4 pilares core | 4 pilares + 2 wrappers top + 6 capacidades base |
| Org chart | menção genérica | 12 pessoas nominais + papéis em 2 slides |
| Reconhecimentos | 3 cards | 6 cards color-coded distintos |
| Atributos pilares | footnote | 7 cards dedicados |
| Multimesas | 8 mesas + AuM no rodapé | 4 métricas AuM dedicadas + 8 mesas + indicadores GD/E |
| ESG | 7 produtos no slide ESG | 8 produtos em slide dedicado + chart histórico Chart.js + práticas + selos |
| Contatos | meta inline pobre | 4 cards (Telefone/SAC/Ouvidoria/Deficientes) + Caixa Postal + ANBIMA + URL |
| Glow radial | Ausente | `--glow-hero` warm + `--glow-cool` no body::before/after |
| Hero blur | Só fade+translateY | filter:blur(16px)→0 (heroIn) |
| Depth tiers | Genérico | `.pillar-card`, `.card-d--*`, `.person-card--hero`, `.mesa` formalizados |
| Color variety | Só accent | 6 cores semânticas (accent/info/success/sage/teal/plum) usadas em coding distinto |
| Stagger | Hardcoded `--anim-i` | Universal via `style="--i: N"` |
| Tipografia | `clamp()` só em deck | `clamp()` em tudo (--size-* tokens) |
| Easing | `cubic-bezier(0.22, 1, 0.36, 1)` em 2 props | `--ease-emphatic` cinematográfico em hero + slide; 4 easings nomeados |
| Letter-spacing | -0.035em | -0.055em no title-mega + variações por escala |
| Brand mark fixo | Ausente | `.brand-mark` top-right + slide-num top-left |
| Deck progress | Ausente | Barra fixa no bottom acima do HUD com glow |
| Cores no chart | Hex genérico | Lidas via `getComputedStyle(--color-accent)` → tema dinâmico |
| Tabela ESG | Estática | Chart.js bar+line responsivo a tema (re-render no toggle) |

## Como abrir

`demos/itau-asset-institucional/output.html` em qualquer navegador moderno.

**Controles:**
- **◀ ▶ / Setas / Space / PageUp-Down** — navegar
- **Home / End** — primeiro / último slide
- **F** — fullscreen (Esc para sair)
- **◐** (HUD) — alternar tema dark/light (Chart.js re-renderiza com cores do tema)
- **#s5** no URL — deep-link para slide 5
